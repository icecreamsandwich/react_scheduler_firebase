## Getting Started


### Prerequisites

* https://github.com/nodejs/node
* https://www.npmjs.com/get-npm


### Installing

* Install node and npm
* Navigate to root project directory
* Install all the packages
```
npm install
```
Development enviroment

```
npm start
```

Deployment build

```
npm run build
```
